#+title: Runtime v0.1 C Bindings

This directory owns the installed pre-v1 C control-lane ABI.  It contains no
graph-plan compatibility API and exposes no Gerbil, Gambit, Python, or Scheme
heap type.

| Owner | Path |
|-------+------|
| Public C11 header | =include/poo_flow/runtime_v0.h= |
| Control contract | =schema/runtime_v0_control.org= |
| Handle/lifecycle engine | =src/runtime_v0.c= |
| Pure C harness | =tests/runtime_v0_harness.c= |

The Bundle remains the semantic owner.  The C engine owns negotiation, opaque
handles, lifecycle state, checkpoint envelopes, owned bytes, and typed boundary
errors only.

Run the focused gates:

#+begin_src sh
make -C bindings/runtime-c clean test sanitize
#+end_src

The installed contract is ABI v0.1 and Bundle schema
=poo-flow.organization-bundle.draft.3=.  Neither is an ABI v1 freeze claim.
