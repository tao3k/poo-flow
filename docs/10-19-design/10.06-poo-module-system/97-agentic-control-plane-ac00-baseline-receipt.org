* Frozen First Vertical Slice

AC-01 is frozen to the smallest scenario accepted in RFC 94:

1. One parent =Agent= delegates exactly one restricted tool capability to one
   child =Agent=.
2. Parent and child each have a typed =Principal= and =Role=.
3. The child's authority is a strict subset of the parent's delegable
   authority; hierarchy creates no grant.
4. The child receives a filtered context projection excluding data outside its
   declared visibility.
5. The child executes one typed external tool effect.
6. Start, allow, deny, timeout, retry, cancellation, checkpoint, restore, stale
   epoch, and evidence outcomes are explicit.
7. Every transition and receipt binds to one deterministic Bundle identity,
   digest, and epoch.

AC-01 produces only the semantic and normalized contract form needed to encode
and validate this scenario.  It does not implement C ABI transport, Python
execution, Cedar evaluation, Lean theorems, or the remaining organization
facets.
