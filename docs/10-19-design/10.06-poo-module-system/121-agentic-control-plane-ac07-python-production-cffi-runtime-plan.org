#+title: AC-07 Python Production CFFI Runtime Implementation Plan
#+date: 2026-07-12

* Delivery

Implement RFC 120 as one path-scoped feature.  Runtime v0 remains unfrozen.
Unrelated CLI, build-cache, and CI work stays outside the commit.

* Step 1 — Public producer amendment

- Replace the private single-event publish hook with public atomic
  =publish_batch= request/result descriptors.
- Validate fixed stride, count arithmetic, monotonic sequence, arena generation,
  payload bounds, queue capacity, and reserved fields before mutation.
- Migrate C harnesses and benchmarks; remove the private header.
- Run strict C, UBSan, installed-consumer, leak, and performance gates.

* Step 2 — Generated CFFI contract

- Add CFFI as a build dependency.
- Generate the out-of-line API declaration from the installed public header;
  reject stale checked-in projections.
- Compile/link against the qualified runtime v0 library with package-relative
  loader paths.

* Step 3 — Native loading and errors

- Add internal =_native= loader, typed errors, status mapping, ABI/layout health
  probe, and explicit library override.
- Keep all native symbols private to the package.
- Fail closed in production; reference mode requires explicit selection.

* Step 4 — Resource owners

- Implement instance, profile, Bundle, session, arena, and lease owners with
  idempotent close and context-manager behavior.
- Hold strong references for =ffi.from_buffer()= arenas.
- Accept caller-provided formal Bundle descriptors; never derive them from graph
  receipts.

* Step 5 — Python runtime integration

- Keep =RuntimeGraphProgram= as the graph-domain execution owner.
- Attach the native adapter through =RuntimeGraphRuntime= context construction.
- Preserve graph materialization and =RuntimeReceipt.plan_digest= unchanged.
- Bind native lifecycle/receipt metadata around execution without inventing C
  event-kind semantics.

* Step 6 — Qualification

- Add contract, loader, error, lifecycle, buffer, producer, differential, and
  public-surface tests.
- Build and install a platform wheel without Gerbil; test bundled and override
  library paths.
- Run the retained 176-test Python suite, AC-01 through AC-07 Gerbil scenarios,
  C sanitizer/leak/install gates, and the actual-wheel performance matrix.
- Write the qualification receipt with =abi-v1-frozen? #f= and commit only when
  the complete feature is green.
