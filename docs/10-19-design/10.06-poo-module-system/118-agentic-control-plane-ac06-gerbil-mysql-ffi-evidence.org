#+title: AC-06 Gerbil and gerbil-mysql FFI Ownership Evidence

* Status

Recorded from the local =.data/gerbil= and =.data/gerbil-mysql= checkouts on
2026-07-12.  This is design evidence, not an ABI v1 freeze receipt.

* Exact owners

| Concern | Owner | Evidence |
|---------+-------+----------|
| Low-level MySQL FFI | =.data/gerbil-mysql/db/_mysql.scm= | =c-define-type= and =define-c-lambda= declarations |
| Public database lifecycle | =.data/gerbil-mysql/db/mysql.ss= | query/result scopes and =finally= cleanup |
| Gerbil foreign ownership model | =.data/gerbil/doc/guide/ffi.md= | owned, borrowed, shallow, array, and custom release types |

* Findings

1. =MYSQL*= has a native release function, while =MYSQL_STMT*= is terminated by
   =mysql_stmt_close=.
2. =MYSQL_RES*= and =MYSQL_FIELD*= are borrowed together and are terminated by
   =mysql_free_result= rather than individually freed.
3. Public query and metadata paths close =MYSQL_RES*= in =finally= blocks.
4. =MYSQL_BIND*= and primitive pointers use explicit native allocation/release.
5. Gerbil =define-c-struct= distinguishes owning pointers, borrowed pointers,
   shallow pointers, arrays, string-member cleanup, and custom release.
6. =char-string= paths rely on string conversion semantics and are unsuitable
   for the production hot payload path.

* ABI consequences

- Arena ownership is explicit and cannot be inferred from =const= or pointer
  type.
- Borrowed slices require a generation-bound lease and one named invalidation
  event.
- One owning arena/resource handle releases the region; item slices are never
  individually freed.
- Hot headers use offsets and lengths, never per-item pointers.
- Production payload bytes remain native and avoid =char-string= conversion.
- Scheme/Gerbil foreign objects are not stable public zero-copy buffers.
- Authorization-covered arena bytes remain immutable until terminal
  acknowledgement.

* Removal and freeze gate

Any future optimization that exposes a Scheme heap pointer, hides result-owner
release, converts each field through =char-string=, or recycles a borrowed region
before lease termination contradicts this evidence and blocks ABI v1 freeze.
